# Playlist Boutique

A single-page website built with Vite and Svelte, deployed to GitHub Pages.

## Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Deployment

The site automatically deploys to GitHub Pages when you push to the `main` branch via GitHub Actions.

**Important:** Make sure to update the `base` path in `vite.config.js` to match your repository name if it's different from `playlist-boutique`.
